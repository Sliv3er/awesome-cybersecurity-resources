## 🛠️ Essential Tools

### 🔍 Reconnaissance
- **[Nmap](https://nmap.org/)** — Network scanner and port mapper
- **[Shodan](https://www.shodan.io/)** — Search engine for Internet-connected devices
- **[theHarvester](https://github.com/laramies/theHarvester)** — OSINT email, subdomain, and IP harvester
- **[Recon-ng](https://github.com/lanmaster53/recon-ng)** — Full-featured web reconnaissance framework
- **[Amass](https://github.com/owasp-amass/amass)** — Network mapping and external asset discovery

### 🌐 Web Application Testing
- **[Burp Suite](https://portswigger.net/burp)** — Web vulnerability scanner and proxy
- **[OWASP ZAP](https://www.zaproxy.org/)** — Open-source web app security scanner
- **[SQLMap](https://sqlmap.org/)** — Automatic SQL injection tool
- **[Nikto](https://github.com/sullo/nikto)** — Web server scanner
- **[ffuf](https://github.com/ffuf/ffuf)** — Fast web fuzzer written in Go

### 🔓 Exploitation
- **[Metasploit](https://www.metasploit.com/)** — Penetration testing framework
- **[Cobalt Strike](https://www.cobaltstrike.com/)** — Adversary simulation and red team ops
- **[John the Ripper](https://www.openwall.com/john/)** — Password cracker
- **[Hashcat](https://hashcat.net/)** — Advanced password recovery
- **[Hydra](https://github.com/vanhauser-thc/thc-hydra)** — Network login cracker

### 🛡️ Defense & Monitoring
- **[Wireshark](https://www.wireshark.org/)** — Network protocol analyzer
- **[Snort](https://www.snort.org/)** — Network IDS/IPS
- **[Suricata](https://suricata.io/)** — High-performance IDS/IPS/NSM
- **[OSSEC](https://www.ossec.net/)** — Host-based intrusion detection
- **[Wazuh](https://wazuh.com/)** — Security monitoring platform

### 🔐 Cryptography
- **[CyberChef](https://gchq.github.io/CyberChef/)** — Swiss army knife for data operations
- **[OpenSSL](https://www.openssl.org/)** — Cryptography toolkit
- **[GPG](https://gnupg.org/)** — Encryption and signing tool
