# Contributing to Awesome Cybersecurity Resources

Thanks for wanting to contribute! 🎉

## How to Contribute

1. Fork this repository
2. Create a branch: `git checkout -b add-resource`
3. Add your resource in the appropriate category
4. Follow the existing format (table or list)
5. Commit: `git commit -m "Add [resource name]"`
6. Push and open a Pull Request

## Guidelines

- **Relevance**: Only add cybersecurity-related resources
- **Quality**: Resources should be well-maintained and reputable
- **No duplicates**: Check existing entries before adding
- **Format**: Follow the existing table/list structure
- **Descriptions**: Keep descriptions concise (under 100 chars)

## Resource Quality Checklist

- [ ] Is the resource actively maintained?
- [ ] Does it have a good reputation in the security community?
- [ ] Is it free or clearly marked as paid?
- [ ] Does the link work?

## Code of Conduct

Be respectful and constructive. We're all here to learn and share knowledge.
